# 🎮 CS2 Pro Matches Data Pipeline

An automated ETL (Extract, Transform, Load) pipeline for professional Counter-Strike 2 match statistics using the PandaScore API.

## Features
- **Data Ingestion**: Fetches recent pro matches and stores them as raw JSON with timestamps.
- **Robust Logging**: Full tracking of the ingestion process.
- **Environment Safety**: Uses `.env` for API key protection.
